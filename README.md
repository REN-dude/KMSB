# KMSB

KMSB はこのリポジトリで開発するアプリケーション／ライブラリの仮タイトルです。現時点では初期セットアップのみを含みます。

## 目的
- 開発の土台となる初期ファイル（README, .gitignore, .editorconfig）の追加
- 今後の実装に向けた基本方針の共有

## 開発環境
- エディタ: Visual Studio / Visual Studio Code
- OS: Windows 10/11
- Git: v2 以降

## はじめかた
1. リポジトリをクローン
2. ブランチを作成して開発
3. 変更をコミット・プッシュ

```
git checkout -b feature/your-task
git add .
git commit -m "feat: your change"
git push -u origin feature/your-task
```

## コーディング規約（暫定）
- インデントは半角スペース 4
- 文字コードは UTF-8（BOM なし推奨）
- 改行コードは Windows (CRLF)
